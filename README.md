This is made for the purposes of galactic photometry. 
Configure settings in parameters in order to define the galaxy you want to observe and pick suitible constraints in the star detector. 
There are some hardcoded values for the dust etc in the galactic profile plot function. These can be changed and edited as you need. 
There are plans to include a better way for this to be doen (ie a .dat file reader) but this is the situation. Make note of the units (especially in the mass to light ratio
comments)
